------------------------------------------------------------------------------------------------------------------
EDITAR CÓDIGO
------------------------------------------------------------------------------------------------------------------

Borrar línea en la que estás 
SHIFT + Supr
CTRL + SHIFT + K

Duplicar una línea
SHIFT + Alt + ↓

Mover bloque de código 
ALT + ↑/↓

------------------------------------------------------------------------------------------------------------------
SELECCIÓN DE CÓDIGO
------------------------------------------------------------------------------------------------------------------

Seleccionar textos iguales y editarlos todos juntos
CTRL + D + D + D...

Seleccionar toda la línea actual
CTRL + L

------------------------------------------------------------------------------------------------------------------
COMENTAR CÓDIGO
------------------------------------------------------------------------------------------------------------------

Comentar líneas de código 
CTRL + K + C

Descomentar líneas de código 
CTRL + K + U

Comentar código seleccionado
ALT + SHIFT + A

------------------------------------------------------------------------------------------------------------------
MOVERTE POR EL CÓDIGO
------------------------------------------------------------------------------------------------------------------

Posicionarte en la línea posterior 
CTRL + ENTER 

Posicionarte en la línea anterior
CTRL + ENTER + Shift 

------------------------------------------------------------------------------------------------------------------
VISUAL STUDIO CODE
------------------------------------------------------------------------------------------------------------------

Abrir buscador para navegar en archivos 
CTRL + P

Crear shortcut de teclado 
CTRL + SHIFT + X

Crear archivo si no existe
Te posicionás con el mouse sobre el nombre del archivo + CTRL. 
Te pregunta si querés guardar uno con ese nombre.

------------------------------------------------------------------------------------------------------------------
FUNCIONALIDADES ÚTILES
------------------------------------------------------------------------------------------------------------------

Moverse por ventanas
ALT + TAB

Cambiar nombre de un archivo
F2
